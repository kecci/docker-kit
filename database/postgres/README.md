# Docker Postgres

## Run Docker compose
```sh
$ docker-compose up -d
```

## Setup server

1. Create server
2. Set host with: `host.docker.internal`
3. Submit

Docs host for local: https://docs.docker.com/desktop/networking/#i-want-to-connect-from-a-container-to-a-service-on-the-host
